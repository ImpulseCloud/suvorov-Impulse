# suvorov-bot
Starcraft 2 Terran bot.

## Build instructions
```bash
# Generate a Makefile.
# Use 'cmake -DCMAKE_BUILD_TYPE=Debug .' if debuginfo is needed.
$ cmake .

# Build.
$ make

# Launch the bot on the specfied map.
$ ./bin/Suvorov "tmp/Ladder2017Season3/InterloperLE.SC2Map"
```
